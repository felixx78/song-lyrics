# [song lyrics](https://song-lyrics.org/)

Next.js app for translating song lyrics

## Tech Stack

- Next.js
- Typescript
- Tailwind CSS
- Zustand
- React Query

## Run Locally

Clone the project

```bash
  git clone https://github.com/felixx78/song-lyrics
```

Go to the project directory

```bash
cd song-lyrics
```

Create .env file

Add

```bash
GENIUS_SECRET=your_token
```

Install dependencies

```bash
npm install
```

Run

```bash
npm run dev
```
